## Descrição
Aplicação Web em ReactJS e NodeJS (PostgreSQL, Sequelize, JWT Authentication, WebRTC e Socket.IO) para live streaming, chat e autenticação prévia dos usuários.

## Funcionalidades
* Live Video/Audio.
* Autenticação JWT.
